# 🛡️ Israel SafeWay App
**Real-time emergency navigation and shelter finder.**

SafeWay helps users navigate Israeli cities while ensuring they are always within 90 seconds (or a specific radius) of a public shelter.

## 🚀 Features
- **Smart Geolocation:** Instant user positioning with a 1km safety radius visualizer.
- **GovMap Integration:** Uses official Israeli Government data for shelter locations.
- **Mock Mode:** Full development environment support using simulated shelter clusters.
- **Safety Routing (WIP):** Calculates paths from A to B that prioritize proximity to protected spaces.

## 🛠️ Tech Stack
- **Frontend:** React + Vite + TypeScript
- **Mapping:** Leaflet.js
- **Data Source:** GovMap API
- **Styling:** CSS3 with custom pulsing animations
